Đăng nhập GitHub API

Cung cấp thông tin repository và Personal Access Token (PAT) để cho phép Admin lưu nội dung trực tiếp vào file data.json trên GitHub. Vercel sẽ tự động deploy lại sau mỗi lần lưu.

Chỉnh sửa trực tiếp dữ liệu dạng JSON. Đảm bảo đúng cú pháp (không thừa dấu phẩy, dùng ngoặc kép).
